# Exercices Java – Boucles
Ce dépôt contient des exercices simples en Java pour apprendre les bases du langage.

## Exercice : Nombres pairs

### Objectif
- Comprendre l'utilisation des boucles en Java
- Utiliser une condition pour tester si un nombre est pair

### Consigne
Afficher tous les nombres pairs entre 1 et 100 :
- une version avec une boucle `for`
- une version avec une boucle `while`

### Contenu
- `NombresPairsAvecFor.java` : solution avec une boucle `for` 
- `NombresPairsAvecWhile.java` : solution avec une boucle `while` 

### Technologies utilisées  
- Java  
- IDE : IntelliJ 

### Niveau
Débutant en Java (bases du langage)
